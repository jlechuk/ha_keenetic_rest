## Keenetic REST API интеграция для Home Assistant

## Описание

Компонент для взаимодействия с роутерами [Keenetic](https://keenetic.ru) из Home Assistant.
Для доступа достаточно пользователя с правом "Веб-конфигуратор".

## Возможности

### Роутер
- Мониторинг системы (CPU, память, скорость приёма/передачи WAN)
- Скорость приёма/передачи WAN интерефейса

### Сетевые клиенты
- Статус подключения
- Скорость приёма/передачи
- Управление регистрацией и доступом в Интернет
- Автоматическое добавление новых сетевых клиентов
- Автообновление имени устройства

## Установка
### HACS
1. Открыть HACS
2. Нажать на 3 точки в правом верхнем углу
3. Выбрать "Пользовательские репозитории"
4. Добавить https://github.com/VVashF/ha_keenetic_rest в "Репозиторий" (Тип -> Интеграция)
5. Перезагрузить Home Assistant

### В ручную
1. Скачать последний релиз
1. Скопировать директорию ha_keenetic_rest в config/custom_components
2. Перезагрузить Home Assistant
